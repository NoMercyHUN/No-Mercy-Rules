DayZ Server Rules

Fair play. Clean gameplay. Clear consequences.

Cheating and exploits

  • Use of cheats, hacks or scripts results in an immediate permanent ban.
  • Exploiting bugs or glitches is forbidden.
  • Any external software providing unfair advantage is forbidden.

Combat and behaviour

  • Combat logging is forbidden.
  • Intentional server disruption is forbidden.
  • Admin impersonation is forbidden.

Base and raid

  • Base glitches or looting through walls are forbidden.
  • Raid rules follow the current server settings.

Reporting

  • Report incidents through a ticket.
  • Provide the time of the incident.
  • Attach video or image evidence if available.
