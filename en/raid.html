Raid Rules

The raid system aims to ensure fair gameplay and maintain server stability.

Basic raid rules

  • Raiding is only allowed through intended server mechanics.
  • Using exploits, bugs or glitches during raids is forbidden.
  • Entering bases through walls, objects or game errors is forbidden.
  • Bypassing base protection mechanics is forbidden.

Forbidden raid methods

  • Wall glitches, boost exploits or texture abuse.
  • Using server restarts to gain raid advantage.
  • Raiding in safe zones or near spawn areas (if such zones exist).
  • Combat logging during raids.

Fair play

  • Intentional harassment or griefing of other players is not allowed.
  • Any activity that threatens server stability during raids is forbidden.
  • Admin impersonation or pretending to be staff is forbidden.

Evidence and reporting

  • Rule violations can be reported via a ticket.
  • Attach video or image evidence if possible.
  • Provide the time of the incident and your Steam64 ID.
